§1 TabICL 能否替代 v2

整孔冷启动：TabICL 在每个指标追平或超过 v2

结论：回归 RMSE 13.86 vs 14.25、R² 0.797 vs 0.786、分类 +1.9pt——TabICL 全面 ≥ v2。 加上许可（v2 禁商用、TabICL BSD-3），v2 已移除，TabICL 成为唯一参考基线。

§3 土层分类

岩性可预测性：几个锚点即大幅内插

冷启动 +3 锚点

冷启动 ~48%（土型难从物性反推），给 3 个散布深度的土型标签后跳到 ~84%。因为岩性在深度上 分段常数（层状），锚点之间几乎是内插填空——这印证了 geo-SCM 先验的分层设计假设。

一句话：TabICL 全面替代 v2（性能 ≥ 且许可可商用）；geo-PFN 验证了"两段式度量利用钻孔相似性" 的机制（锚点增益最优、零信息冷启动追平），但绝对精度受限于合成先验的真实度——这是明确、可修的下一步。
